# EclipseScraper

Scrape NASA's ecliupse website to generate EclipseTracks.org CZML and JSON files.

## Setup

From `~/eclipsescraper`:

```
poetry shell
poetry install --with dev
```

## Run the Scraper

From `~/eclipsescraper`:

```
python eclipsescraper.py
```

## Run Tests

From `~/eclipsescraper`:

```
pytest tests
```
